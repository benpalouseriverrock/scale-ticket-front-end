Expected calculations:
  Input:
    - Gross Weight: 70000 lbs
    - Truck Tare: 30240 lbs
    - Trailer Tare: 8000 lbs
    - Delivery Method: location (Colfax)
    - Delivery Charge: $10.00
    - Product Price: $5.50/ton
    - Tax Rate: 7.9% (WA)

  Auto-Calculated:
    - Net Weight: 70000 - (30240 + 8000) = 31760 lbs
    - Net Tons: 31760 / 2000 = 15.88 tons
    - Material Cost: 15.88 × $5.50 = $87.34
    - Subtotal: $87.34 + $10.00 = $97.34
    - Tax Amount: $97.34 × 7.9% = $7.69
    - Total: $97.34 + $7.69 = $105.03

Open browser console (F12) → Network tab → Click Print button to see exact values

================================================================================
                      TROUBLESHOOTING
================================================================================

ERROR: "Cannot connect to backend"
FIX:
  1. Check backend is running: npm run dev (in backend folder)
  2. Verify apiUrl in src/environments/environment.ts
  3. Check backend is on port 3000
  4. Check firewall allows localhost:3000

ERROR: "Module not found" or "import errors"
FIX:
  1. Run: npm install
  2. Delete node_modules folder
  3. Run: npm install again

ERROR: "Dropdown options don't appear"
FIX:
  1. Check backend API is responding
  2. Open browser console (F12) → Network tab
  3. Look for GET /api/customers, /api/products, etc.
  4. If 404 errors, check backend routes are implemented

ERROR: "Can't create ticket"
FIX:
  1. Check all required fields are filled
  2. Open console and look for error message
  3. Check Network tab for POST /api/tickets response
  4. If 500 error, check backend logs

ERROR: "Calculations don't match expected values"
FIX:
  1. Check backend ticketController.js calculation logic
  2. Check all referenced tables have correct data
  3. Check tax_rate for customer's state
  4. Check delivery_rates table has correct rates
